# 🎮 Desktop Games — 上班摸鱼放松小游戏

> ⏰ 老板来了记得按 `Ctrl+W` · 摸鱼专用 · 网页即开即玩

两个经典小游戏，专为 **上班摸鱼** 设计：
- ✅ 无需安装，双击 HTML 即开即玩
- ✅ 极低 CPU 占用，轻松隐藏
- ✅ 支持键盘 / 触屏，手机电脑都行
- ✅ 设置持久化，下次打开还是你的配置

---

## 🐍 贪吃蛇 Snake Game

经典贪吃蛇，吃食物变长，撞墙或自撞则游戏结束。

**特色功能：**
- 🎨 **彩虹渐变色蛇身** — 蛇越长颜色越绚烂，头部有眼睛会跟着方向转
- 🌈 **三种食物** — 普通🔴(1分)、金色⭐(3分)、翡翠💎(5分)，稀有食物带发光特效
- 🧱 **穿墙模式** — 从一侧穿到另一侧，更轻松
- ⚡ **四档速度** — 🐢休闲 / ⚡普通 / 🔥快速 / 💀地狱
- 🏆 **最高分记录** — 保存在浏览器中，挑战自己

**操作：**
| 按键 | 功能 |
|------|------|
| `↑↓←→` / `WASD` | 控制方向 |
| `Space` | 暂停 / 继续 |
| 点击「开始游戏」 | 重新开局 |
| 触摸滑动 (手机) | 控制方向 |
| `R` | 重置游戏 |

---

## 🧱 俄罗斯方块 Tetris

经典俄罗斯方块，消除行数得分，越玩越快。

**特色功能：**
- 👻 **预测线** — 显示方块落点位置（可关闭增加难度）
- ⬇ **下一个方块预览** — 提前规划摆放策略
- 📦 **五档初始等级** — 🍼新手 / ⚡普通 / 🔥高手 / 💀地狱
- 🧱 **完整 7 种方块** — I / O / T / S / Z / J / L
- 🔄 **墙踢 (Wall Kick)** — 贴墙也能旋转，手感流畅
- 🏆 **最高分记录** — 持久化保存

**操作：**
| 按键 | 功能 |
|------|------|
| `← →` / `A D` | 左右移动 |
| `↑` / `W` | 旋转方块 |
| `↓` / `S` | 加速下落 |
| `Space` | 硬降到底 |
| `P` | 暂停 / 继续 |
| 点击「开始游戏」 | 重新开局 |
| 触摸点击 (手机) | 旋转 |
| 触摸滑动 (手机) | 移动 / 硬降 |
| `R` | 重置游戏 |

## ⚙ 统一设置

两个游戏右上角都有 **⚙ 齿轮按钮**，随时调整：

**贪吃蛇设置：**
- 移动速度（4档）
- 地图边界（有边界 / 穿墙）

**俄罗斯方块设置：**
- 初始等级（影响下落速度）
- 预测线（显示 / 隐藏）

设置自动保存到浏览器，下次打开不用再调。
